**ChronoContext** решает проблему анахронизмов в ролевых играх. Персонаж в 2003 году листает TikTok. В 1995-м расплачивается Apple Pay. В 2010-м обсуждает сериал, который выйдет через пять лет.
*Расширение автоматически собирает данные из Wikipedia, обрабатывает их через AI и инжектит готовый «пакет эпохи» в системный промпт. AI-модель начинает вести себя так, будто действительно живёт в выбранном году.*

ChronoContext автоматически генерирует подробный контекстный пакет для выбранной страны и года, включающий:

- Какие технологии существуют (и какие **нет**)
- Что показывают в кино, что играет по радио, во что играют на компьютерах
- Сколько стоит хлеб, какая средняя зарплата, какой курс доллара
- Как люди общаются, знакомятся, развлекаются
- Какой сленг используют
- Чёрный список анахронизмов: вещи, которые AI **обязан** игнорировать

Пакет инжектится прямо в системный промпт SillyTavern. Никаких ручных правок, никакого копирования текстов.

## ✨ Как начать

**1.** Открой панель расширений → разверни **ChronoContext**

**2.** Выбери страну и год, нажми **«Найти и сгенерировать пакет»**

**3.** Поставь галочку **«Включить инжекцию в контекст»**

Готово. AI-модель теперь знает, что в 2003-м в России нет ни Telegram, ни TikTok, ни Яндекс.Еды, зато есть ICQ, кнопочные Nokia и сериал «Бригада» по телевизору.

## ⚙️ Настройки

### Страна

Выпадающий список из 19 предустановленных стран. Для экзотических вариантов выбери «Другая» и введи название на **английском языке** (например, `Ottoman Empire`, `Czechoslovakia`, `Byzantine Empire`).

### Год

Слайдер: 1800–2025. Числовое поле: 1–2025. Для древних периодов (до 1800) используй числовое поле.

> ⚠️ Чем дальше в прошлое, тем меньше данных в Wikipedia. Для периодов до 1900 года AI будет опираться в основном на собственные знания.

### Язык пакета

Язык, на котором AI сгенерирует финальный пакет. Рекомендуется выбирать язык ролевой игры.

| Код | Язык |
|---|---|
| `ru` | Русский |
| `en` | English |
| `ja` | 日本語 |
| `zh` | 中文 |
| `ko` | 한국어 |
| `es` | Español |
| `fr` | Français |
| `de` | Deutsch |

### Инжекция

Галочка «Включить инжекцию в контекст» управляет тем, вставляется ли пакет в промпт. Можно сгенерировать пакет, посмотреть превью, отредактировать, и только потом включить.

## 👁️ Превью и редактирование

После генерации нажми **«Превью пакета»**, чтобы увидеть содержимое.

Для ручного редактирования:
1. Нажми **«Редактировать вручную»**
2. Внеси изменения в текстовом поле
3. Нажми **«Сохранить правки»**

Ручные правки перезаписывают кэш. Чтобы отменить правки, нажми «Отменить» до сохранения.

## ❓ FAQ

**Q: Сколько времени занимает генерация?**
A: От 30 секунд до 2-3 минут. Зависит от скорости Wikipedia API и твоей AI-модели.

**Q: Нужно ли генерировать пакет каждый раз?**
A: Нет. Пакет кэшируется на 72 часа. При повторном запросе с теми же параметрами загрузка мгновенная.

**Q: Можно ли использовать для фэнтези-миров?**
A: ChronoContext заточен под реальную историю. Для вымышленных миров лучше написать контекст вручную и вставить через редактирование пакета.

**Q: Можно ли иметь несколько пакетов одновременно?**
A: Активным может быть только один пакет. Но кэш хранит все сгенерированные пакеты, и переключение между ними мгновенное (сгенерируй новый — старый останется в кэше).
